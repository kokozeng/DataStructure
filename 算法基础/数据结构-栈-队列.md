# 栈

先进后出

```c++
#include <iostream>

using namespace std;

const int N = 1e6 + 10;

int stk[N], tt;

// 插入

stk [ ++ tt] = x;

// 弹出

tt --;

// 判断是否为空

if (tt > 0) 不空
else 空

// 栈顶

stk[tt];
```



# 队列

先进先出

```c++
#include <iostream>

using namespace std;

const int N = 1e6 + 10;

int q[N], hh, tt = -1; // tt的边界看个人喜好

// 在队尾插入元素

q[++ tt] = x;

// 在队头弹出元素

hh ++; // 队头指针往后移动一个就弹出了

// 判空

if (hh <= tt) not empty
else empty

// 取出队头元素

q[hh]；

//还能取队尾
```
